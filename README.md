# Anotações

## Version
```
python 3.8.0
```

## Setup venv

```
source ./scripts/setup-venv.sh
```

ou

```
scripts\setup-venv
```

## Executar projeto

```
source ./scripts/execute.sh
```

## Formatar arquivo python
```
autopep8 -iv src/main.py
```
